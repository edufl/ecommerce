# 🛒 Fake E-commerce (React)

Projeto de e-commerce desenvolvido em React com funcionalidades reais de uma loja online.

---

## 🚀 Funcionalidades

- 🔍 Listagem de produtos (API)
- 🧩 Filtro por categoria
- 🔎 Busca por nome
- 🛒 Carrinho de compras
- ➕ Controle de quantidade
- 💳 Checkout com pagamento fake
- 🔐 Login fake (autenticação)
- 🔔 Toast de notificações
- 📱 Responsivo (mobile)

---

## 🛠️ Tecnologias

- React
- React Router
- Context API
- JavaScript (ES6+)
- CSS3
- API (Fake Store / DummyJSON)
